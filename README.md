# ZLDR Events

An Android app for members of **ZLDR (Zwift Long Distance Runners and Riders)** that displays all upcoming ZLDR cycling and running events directly from Zwift.

## Download

Go to the [**Releases**](https://github.com/victorypoint/ZLDREvents/releases) page to download the latest APK and user guide.

## Requirements

- Android 8.0 (Oreo) or later
- A Zwift account

## Installation

1. Download `ZLDREvents-1.2.0.apk` from the [Releases](https://github.com/victorypoint/ZLDREvents/releases) page
2. On your Android device, enable **Install from unknown sources** (Settings → Security, or Settings → Apps → Special app access → Install unknown apps)
3. Open the downloaded APK and tap **Install**
4. Sign in with your Zwift username and password

## About

ZLDR Events is a read-only viewer — it does not join events, modify your Zwift profile, or store any personal data beyond your Zwift login session.

Developed by Alan Udell for the ZLDR community.
